# 雰文典

本書では、人工言語「雰語」の文法事項について解説する。

本書は雰語の文法事項を簡潔に列挙し整理したものである。雰語学習者のための教本としては（準備中…）を推奨する。また、ここで扱う「雰語」とは、特に注記のない限り第三版の現代標準雰語を指す。

## 表記について

本書内では、原則的に漢字転写と符号ラテン文字を用いて雰語を表記する。また、本文中で雰語部分とそれ以外を区別するために、前者をセリフ体、後者をサンセリフ体で表記する。
