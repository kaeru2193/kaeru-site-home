# 文字と転写

雰語では、主に**雰字**と呼ばれる表語文字が書記に用いられます。

<div className="phun text-3xl">
穏永！ 我為之機。
現書之言以栄字。
</div>
↑雰字で書かれた雰語文の例。

ただし雰字の使用はコンピューター上での扱いや学習面などで不便な場合があるため、いくつかの転写方法が定められています。

## 漢字転写

雰字一字づつを字義が類似する漢字に転写する方法です。例として、上の例文を漢字転写したものがこちらです。

```
穏永！ 我為之機。
現書之言以栄字。
```

詳細は後の講座で説明していきます。

## ラテン文字転写

雰語の発音をラテン文字に転写する方法です。雰字の読みを記述する場合に用いられます。再び例として、上の例文を転写したものがこちらです。

```
wáihès! xê xêl kúaàil.
váim sûa kúa bùn nâo fúnqô.
```

声調部分を数字で書く場合もあります。

```
wai1 hes2! xe3 xel3 kua1 ail2.
vaim1 sua3 kua1 bun2 nao3 fun1 qo3.
```

こちらも詳細は後の講座で説明していきます。
