# 音韻論

## 音節

雰語の音節は明確な構造を有し、「(C)V(X)<sup>T</sup>」という形で記述することが可能である。ここで C は頭子音、 V は母音、 X は末子音、 T は声調を指す。括弧で囲んだ要素は存在しない場合もある。

## 頭子音

雰語では20の頭子音が区別される。左側がラテン文字転写、右側がIPA表記である。

<span className="ipa">/h/</span>の調音は話者によって様々であるが、狭母音 (<span className="ipa">/i/</span>, <span className="ipa">/u/</span>) の前では<span className="ipa">[x]</span>、それ以外の母音の前では<span className="ipa">[h]</span>として実現される傾向にある。

## 声調

雰語は声調言語であり、3つの声調が区別される。転写例では便宜上 <PText>a</PText> を例に取る。

<ScrollableImg src="/phun-grammar/tones.png" height={400}/>

### 第一声

高く平らな声調。調値は <span className="ipa">/⁵⁵/</span>。ラテン文字転写では <PText>á</PText> 、数字表記では <PText>a1</PText> と表記される。

### 第二声

低く平らな声調。調値は <span className="ipa">/²²/</span>。ラテン文字転写では <PText>à</PText> 、数字表記では <PText>a2</PText> と表記される。

但し、第二声が2つ以上連続する場合には末尾以外の音節の声調が<span className="ipa">/²⁴/</span>に変化する。これは語を跨いで第二声が連続する場合にも発生する。

### 第三声

高くから下降する声調。調値は <span className="ipa">/⁵³/</span>。ラテン文字転写では <PText>â</PText> 、数字表記では <PText>a3</PText> と表記される。

## 音配列の禁則

雰語の音節のうち、<PText>yi</PText> と <PText>wu</PText> を含むものは禁則であり出現しない。
