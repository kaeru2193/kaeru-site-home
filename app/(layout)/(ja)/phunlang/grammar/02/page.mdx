# 音韻論

## 音節

雰語の音節は明確な構造を有し、「(C)V(X)<sup>T</sup>」という形で記述することが可能である。ここで C は頭子音、 V は母音、 X は末子音、 T は声調を指す。括弧で囲んだ要素は存在しない場合もある。

## 頭子音

雰語では20の頭子音が区別される。左側がラテン文字転写、右側がIPA表記である。

<span className="ipa">/h/</span>の調音は話者によって様々であるが、狭母音 (<span className="ipa">/i/</span>, <span className="ipa">/u/</span>) の前では<span className="ipa">[x]</span>、それ以外の母音の前では<span className="ipa">[h]</span>として実現される傾向にある。

## 声調

雰語は声調言語であり、3つの声調が区別される。転写例では便宜上 <PText>a</PText> を例に取る。

<ScrollableImg src="/phun-grammar/tones.png" height={400}/>

### 第一声

高く平らな声調。調値は <span className="ipa">/⁵⁵/</span>。ラテン文字転写では <PText>á</PText> 、数字表記では <PText>a1</PText> と表記される。

### 第二声

低く平らな声調。調値は <span className="ipa">/²²/</span>。ラテン文字転写では <PText>à</PText> 、数字表記では <PText>a2</PText> と表記される。

但し、第二声が2つ以上連続する場合には末尾以外の音節の声調が<span className="ipa">/²⁴/</span>に変化する。これは語を跨いで第二声が連続する場合にも発生する。

### 第三声

高くから下降する声調。調値は <span className="ipa">/⁵³/</span>。ラテン文字転写では <PText>â</PText> 、数字表記では <PText>a3</PText> と表記される。

## 音配列の禁則

雰語の音節のうち、<PText>yi</PText> と <PText>wu</PText> を含むものは禁則であり出現しない。

## 連音規則

閉音節（末子音のある音節）の後ろに母音または頭子音/j/で始まる音節が続く場合、閉音節の末子音を次の音節が引き継ぐ。特に頭子音 <span className="ipa">/j/</span> で始まる音節が後続している場合には、閉音節の末子音が口蓋化した上で引き継がれる。

またこの連音により、雰拼での表記上は区別される語でも、発音上では区別されなくなる場合があることに注意。

- <PPron text="大弟">大弟</PPron> → <span className="ipa">[t͡ʃim˥˧.mi˨]</span> （発音上は <PText>qîmmì</PText> と区別されない）
- <PPron text="火痛">火痛</PPron> → <span className="ipa">[taŋ˨˦.ŋɯo˨]</span>
- <PPron text="賜善">賜善</PPron> → <span className="ipa">[d͡ʒɛl˨.lɯas˥]</span> （発音上は <PText>jèllúas</PText> と区別されない）
- <PPron text="位処">位処</PPron> → <span className="ipa">[dɯn˨.nʲai˥]</span>
- <PPron text="弱指">弱指</PPron> → <span className="ipa">[sɯl˨.lʲɯo˥]</span>

末子音が無い開音節では主母音が長く発音され、これによって閉音節と開音節が区別される。例えば <PText>jèl'úas</PText> と <PText>jè'lúas</PText> は、母音<span className="ipa">/ɛ/</span>の持続時間によってのみ区別される。

また、末子音 <span className="ipa">/s/</span> に <span className="ipa">/ʃ/</span>, <span className="ipa">/t͡ʃ/</span>, <span className="ipa">/d͡ʒ/</span> のいずれかで始まる音節が後続する場合、末子音 <span className="ipa">/s/</span> は同化して <span className="ipa">[ʃ]</span> と発音される。更に、末子音 <span className="ipa">/s/</span> に <span className="ipa">/z/</span> で始まる音節が後続すると、末子音 <span className="ipa">/s/</span> は同化して <span className="ipa">[z]</span> と発音される。

- <PPron text="律知">律知</PPron> → <span className="ipa">[giʃ˥.t͡ʃai˨]</span>
- <PPron text="建材">建材</PPron> → <span className="ipa">[ʃɛʃ˥.d͡ʒaim˥˧]</span>
- <PPron text="天京">天京</PPron> → <span className="ipa">[βaz˥.za˥˧]</span>
